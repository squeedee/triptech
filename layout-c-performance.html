Concept: Designed for live performance. Filter Freq and Q are always mapped to the two large left encoders — these are the most expressive, most-tweaked parameters. The two right encoders cycle through pages: Amp (Drive/Amount), Envelope (Atk/Dcy), Mixer (Pan/Level), Send (Send/LFO), and Global (Delay/Dry). Channel buttons select which channel all 4 knobs affect.

Hardware: 4 encoders (2 large + 2 small) + 3 channel buttons + 1 page button + 2 transport buttons + 128x32 OLED
Pros: Instant access to filter (most important for performance), only 1 press to change channel, 1 press to cycle context page, very expressive live
Cons: Global params (delay/pattern) require paging to page 5, can only edit one channel at a time
