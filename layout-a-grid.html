Layout A: Grid Select

4 channel buttons x 3 group buttons = instant access to any parameter page. Each encoder has its own 0.91" 128x32 OLED showing parameter name, value, and bar graph. Shift button reveals secondary functions (LFO AMT, SLOPE, SYNC).

Concept: Inspired by Elektron workflow. Channel buttons select which channel you're editing (or Global for delay/LFO). Group buttons select the parameter group. Each of the 3 encoders has a dedicated 0.91" OLED above it showing parameter name, value, and a bar graph. Hold Shift to access secondary functions (LFO AMT, SLOPE, SYNC).

Hardware: 4 channel buttons + 3 group buttons + 3 transport buttons (RUN, BYP, SFT) + 3 rotary encoders + 3x 128x32 OLED
Pros: Systematic, predictable, max 2 presses to any param, per-encoder display gives clear feedback
Cons: 10 buttons + 3 OLEDs, can't tweak two channels simultaneously
