Layout A: Grid Select

4 channel/global keys cycle through parameter groups. 3 LEDs indicate the active group. Each encoder has its own 0.91" 128x32 OLED. Shift reveals secondary functions (LFO AMT, SLOPE, SYNC).
Triptech
Concept: Press a channel key (1/2/3/G) to select it; press again to cycle groups. 3 LEDs show active group. Reference table maps groups to parameter names per channel. Hold Shift for secondary functions (LFO AMT, SLOPE, SYNC).

Hardware: 4 channel keyswitches + 3 function keys (RUN, BYP, SHIFT) + 3 group LEDs + 3 rotary encoders + 3x 128x32 OLED
Pros: Minimal buttons, systematic, max 3 presses to any param, per-encoder display
Cons: Need reference table to know group meanings, can't tweak two channels simultaneously
