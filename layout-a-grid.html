Layout A: Grid Select

4 channel buttons (Ch1/Ch2/Ch3/Global) x 3 group buttons = instant access to any parameter page. 4 encoders edit the current page. Max 2 button presses to reach any parameter.

Concept: Inspired by Elektron workflow. Channel buttons along the left select which channel you're editing (or Global for delay/LFO). Group buttons across the top select the parameter group. The 4 encoders always map to 3-4 parameters from that intersection. A small OLED shows current assignments and values.

Hardware: 4 channel buttons + 3 group buttons + 2 transport buttons + 4 rotary encoders + 128x32 OLED
Pros: Systematic, predictable, max 2 presses to any param, easy to learn
Cons: 9 buttons total (more hardware), can't tweak two channels simultaneously
