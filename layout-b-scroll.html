Layout B: Scroll Page

Minimal hardware. One encoder scrolls through a flat page list, three encoders edit parameters. Push the nav encoder to jump between channel/global sections.
Triptech
PAGE
push: jump section
Concept: Like a digital synth (Blofeld, Hydrasynth). One nav encoder scrolls a flat list of pages. Three encoders edit the 3 parameters on each page. Push the nav encoder to jump to the next channel/section.

Hardware: 4 rotary encoders (1 with push) + 2 transport buttons + 128x64 OLED
Pros: Minimal hardware (6 total controls), deep access to everything, compact physical size
Cons: Slower navigation to distant pages, some pages only have 1-2 params (wasted encoders)
